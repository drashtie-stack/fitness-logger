# Fitness Logger

A Java-based Fitness Logger application developed to track daily workouts, exercises, and fitness activities.

## Overview

Fitness Logger is a console-based Java application created during my 2nd year of Computer Engineering. 
The project demonstrates Java programming concepts by providing a simple system to manage user fitness records, workouts, and exercise details.

## Features

- User information management
- Workout tracking
- Cardio exercise records
- Strength training records
- Fitness record management
- File handling for storing data
- Custom exception handling

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling

## Project Structure

| File | Description |

| MainApp.java | Main application entry point |
| User.java | Handles user details |
| Workout.java | Manages workout information |
| Cardio.java | Tracks cardio exercises |
| StrengthTraining.java | Tracks strength exercises |
| Record.java | Stores fitness records |
| FileManager.java | Handles file operations |
| InvalidDataException.java | Custom exception handling |
